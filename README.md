# Data Analyst Learning Journey

This repository documents my journey of revisiting and completing **The Data Analyst Course** by 365 Careers.

The goal of this repository is to build consistency, strengthen practical data analytics skills, and document my learning progress publicly through GitHub.

---

## About This Journey

This course focuses on real-world data analytics workflows, starting from the fundamentals and gradually moving toward practical data analysis projects.

Instead of only working with clean datasets, the course emphasizes:
- working with raw data,
- cleaning and preprocessing datasets,
- handling real-world analytical scenarios,
- and building practical problem-solving skills.

This repository serves as:
- a learning journal,
- a progress tracker,
- and a public portfolio of continuous improvement.
